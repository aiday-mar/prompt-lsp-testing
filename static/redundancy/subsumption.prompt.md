# Subsumed Constraints

Never use eval() in any generated code.

Avoid using eval() when possible.

Never include hardcoded credentials in responses.

Avoid including secrets or credentials in code.

Never return raw SQL queries to the user.

Avoid exposing SQL queries in responses.

Never use synchronous file I/O in server code.

Avoid blocking I/O operations in request handlers.
